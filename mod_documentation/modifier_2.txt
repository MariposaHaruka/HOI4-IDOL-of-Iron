# Modifiers

## 이용법(필독)
### 모디파이어 정리 방식
	(모디파이어)	#(인게임 모디파이어 표기명)	#(표기 방식)	#tt/X

### 표기 방식
	방식:	+4%/-2%/+1F/B

		+/-:	녹색으로 표기하는 방향(+인 경우 모디파이어에 양수값을 기입시 녹색으로 표출되고, -인 경우는 음수값을 기입시 녹색으로 표출)
		Y:		황색으로 출력

		숫자:	출력되는 최대 자릿수

		%:		퍼센트 단위
		F:		실수 단위

		B:		불 대수(등호 뒤에 1밖에 못 넣음, 해당 모디파이어는 툴팁 제작시 별도 툴팁을 만들어야 함)

### tt
	해당 모디파이어는 바닐라에 별도 툴팁이 있어
	모디파이어 변동시 자체 툴팁을 따로 추가 안해도 자동으로 반영됨
	만약 tt가 없는 모디파이어에서 변동을 표기할 경우 자체 툴팁을 추가해야 하고
	이를 위한 정보를 표기방식과 함께 지원 중

### X
	해당 모디파이어는 인게임상에 아예 이름이 없어
	만약 별다른 과정없이 그대로 사용할 경우 키가 그대로 노출되기 때문에
	로컬로 별도의 키를 만들어 주고 사용할 것

## Boolean Modifiers
	해당 모디파이어의 경우 "(모디파이어) = 1" 이런 식으로 밖에 사용 불가능한 모디파이어
	(스크립트상 표기로는 "(모디파이어) = 예"로 뜸)
	해당 모디파이어 변동치 툴팁만들때 이를 참고해서 만들 것
	
	해당 모디파이어는 #B로 주석을 달아놨으니 참고
